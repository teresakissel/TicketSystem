# Python Banking Support Ticket System
A menu driven Python application that simulates tracking and resolving online banking support issues.

## Features
- Create and manage support tickets
- Update ticket status (Open, In Progress, Resolved)
- Persist ticket data using JSON storage
  
## Technologies Used
- Python
- JSON
- Command Line Interface
  
## Purpose
Desinged to reflect real-world application support and troubleshooting workflows
